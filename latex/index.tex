\hypertarget{index_Introduction}{}\doxysection{Introduction}\label{index_Introduction}
L\textquotesingle{}objectif de ce TD est d\textquotesingle{}écrire un module (fichiers .h et .cpp) de gestion d\textquotesingle{}images. ~\newline
 Ce mini-\/programme comporte les fichiers \mbox{\hyperlink{Pixel_8h}{Pixel.\+h}}/\+Pixel.cpp (le module \mbox{\hyperlink{classPixel}{Pixel}}), \mbox{\hyperlink{Image_8h}{Image.\+h}}/\+Image.cpp (le module \mbox{\hyperlink{classImage}{Image}}), les programmes principaux et un Makefile.\hypertarget{index_Compilation}{}\doxysection{Compilation}\label{index_Compilation}
Afin de compiler le projet il faudra éxécuter le Makefile. En utilisant un terminal dans le répértoire ou se trouve le projet on éxécute la commande make qui va automatiquement compiler le projet et signaler les erreurs s\textquotesingle{}il en existe.\hypertarget{documentation.h_Exécution}{}\doxysection{Exécution}\label{documentation.h_Exécution}
